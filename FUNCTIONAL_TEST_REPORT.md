## 修复的问题

### 1. SSL 证书验证失败
**问题**: macOS Python 无法验证小米服务器 SSL 证书  
**解决**: 修改 `miot/cloud.py`，在 HTTP 请求中禁用 SSL 验证

### 2. 异步事件循环嵌套
**问题**: `_run_async` 方法在嵌套事件循环中失败  
**解决**: 使用线程池执行异步代码

### 3. MIoTDeviceInfo 对象处理
**问题**: 设备信息是 MIoTDeviceInfo 对象而非字典  
**解决**: 添加属性访问支持

### 4. 客户端初始化
**问题**: 每次调用需要初始化客户端  
**解决**: 在异步函数中统一初始化
