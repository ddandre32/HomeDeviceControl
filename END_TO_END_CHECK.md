## 📋 使用说明

### 安装步骤

```bash
# 1. 进入 Skill 目录
cd ~/Documents/openclaw_lab/skills/XiaomiDeviceControl

# 2. 运行安装脚本
./install.sh

# 3. 激活虚拟环境
source ~/Documents/openclaw_lab/xiaomi_env/bin/activate

# 4. 获取认证 URL
miot system oauth-url

# 5. 访问 URL，登录小米账号，获取授权码

# 6. 完成认证
miot system auth <授权码>

# 7. 测试设备列表
miot device list
```

### 使用方法

```bash
# 自然语言控制
python3 skill.py natural_language_control "打开客厅的灯"

# 列出设备
python3 skill.py list_devices

# 控制设备
python3 skill.py control_device "客厅灯" turn_on

# 批量控制
python3 skill.py batch_control '[{"device_name":"客厅灯","action":"turn_on"},{"device_name":"卧室灯","action":"turn_on"}]'

# 执行场景
python3 skill.py execute_scene "回家模式"

# 发送摩斯密码
python3 skill.py send_morse_code "你好" --name "台灯"
```

## ⚠️ 已知限制

3. **传感器触发** - 需要小米传感器设备支持
4. **中文摩斯** - 仅支持部分常用汉字
